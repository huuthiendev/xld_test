# XLD Test

## Setup
Install all dependencies
```bash
npm install
```
Update ENV (.env.example)
```
DB_HOST=<<Database Host>>
DB_PORT=<<Database Port>>
DB_USER=<<Database Username>>
DB_PASSWORD=<<Database Password>>
DB_NAME=<<Database Name>>

APP_PORT=<<App Port>>
```

Rename .env.example to .env

Start app
```bash
npm run start
```
